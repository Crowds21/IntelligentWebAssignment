# IntelligentWebAssignment
- [Github](https://github.com/Crowds21/IntelligentWebAssignment)
- Nodejs(16)
- MongoDB

## Build envrionment
- Install Mongodb.
```shell
git clone https://github.com/Crowds21/IntelligentWebAssignment.git
cd IntelligentWebAssignment
npm install
npm run start
```
